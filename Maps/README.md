# How to Install Maps

For each map copy the following file types to the given destination:

| File Type         | Destination  |
| ----------------- | ------------ |
| *.unr, *.ut2      | Maps         |
| *.utx             | Textures     |
| *.umx, *.ogg      | Music        |
| *.uax             | Sounds       |
| *.u, *.int, *.ini | System       |
| *.usx             | StaticMeshes |
| *.ukx             | Animations   |
| *.ka              | Karma        |
| *.upx             | Prefabs      |
